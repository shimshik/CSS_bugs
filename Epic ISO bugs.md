# **Epic для багов на ISO образы** #

**Описание ошибки:** 
Менеджер сетевых интерфейсов выкидывает ошибку при запуске виртуальной машины с <2 сетевых интерфейсов

**Проект:** vESR

**Серьезность:** S1 Blocker

**Приоритет:** High

**Статус:** New

**Автор:** Шимшилашвили М.И.

**Окружение:** 
Родительская ОС - Windows 11.25H2 (сборка 26200.8246)

Гипервизор - Oracle VirtualBox 7.2.4 r170995 

Гостевая ОС - Other Linux 64-bit

**Шаги воспроизведения:**

1. Создать виртуальную машину используя упаковку vESR 1.28.ova
   
2. В настройках сети виртуальной машины оставить включенным <2 сетевых интерфейсов

3. Запустить виртуальную машину
 
**Ожидаемый результат:** Виртуальная машина запускается и находится в полном работоспособном состоянии, в списках сетевых интерфейсов виртуальной машины отображаются только включенные в настройках интерфейсы

**Фактический результат:** Менеджер сетевых интерфейсов выкидывает ошибку nic_mgr_autosort_ifnames_at_start(&nic_mgr) failed, цикл nic_mgr_autosort_ifnames_at_start уходит в бесконечную рекурсию с такой же ошибкой
